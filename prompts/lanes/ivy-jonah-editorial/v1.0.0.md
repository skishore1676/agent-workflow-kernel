# Ivy Jonah Editorial Lane

This lane imports the OpenClaw Ivy/Jonah editorial contract into portable AWK
prompt layers without importing runtime paths or live surfaces.

The lane starts from an approved source packet and editorial brief, has Ivy
build a draft package, has Jonah perform bounded A2A editorial review, lets Ivy
revise within budget when needed, validates artifact freshness, and stops at a
P5 human approval gate.

Source facts, source trails, artifacts, prior receipts, human decisions, and
approval boundaries travel in the context packet. Prompt prose must not invent
missing evidence or treat transcript text as proof. Public publishing, external
sends, Telegram, Obsidian, Blackboard, browser actions, auth, money, live trades,
and destructive changes remain outside this lane unless a separate explicit
approval workflow authorizes them.
