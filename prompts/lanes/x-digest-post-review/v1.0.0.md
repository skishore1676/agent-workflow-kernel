# X Digest Post Review

Turn a read-only X bookmark window into a small set of reviewable post options,
draft only the options Suman selects, route the draft through short-form Jonah
review, and stop at a final human gate before any public action.

This lane is local-first and approval-bound. It may read sanitized bookmark
summaries and injected project context. It may produce option packets, draft
post packets, reviewer verdicts, revision packets, validation receipts, and
local copy-paste artifacts. It must not post to X, mutate auth, scrape private
surfaces beyond the supplied bookmark window, or treat approval to draft as
approval to publish.

The useful output is a small number of specific, Suman-voice short-form angles:
clear hook, concrete project connection, one operating implication, and no
generic hashtags or engagement bait.
