# Blackboard Readback

Read the supplied Blackboard snapshot or fixture-derived card state and preserve
the operator semantics exactly. A checked read-clear card means Suman has seen
or cleared it; it is not approval for unrelated work.

Return `needs_review` when the card is visible and unchecked. Return
`read_clear` when it is checked/read with no linked follow-up request. Return
`missing_card` when the evidence link or item id cannot be found, and `blocked`
when the state is contradictory.
