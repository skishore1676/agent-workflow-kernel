# Discover Weekly Artifact

Locate the supplied weekly update fixture or declared artifact reference and
summarize its stable identifiers. This stage is read-only: do not inspect live
vault state, write Obsidian, send Telegram, mutate OpenClaw runtime files, or
clear Blackboard cards.

Return `found` when the artifact exposes mode, note path, and source artifact
metadata. Return `no_artifact` when no weekly item is present, and `blocked`
when identifiers are contradictory or unsafe to use.
