# X Digest Final Publish Gate

Pause for Suman's exact final decision on the validated post packet. In the
current tracer, approval materializes a local Obsidian copy-paste packet only.
It does not authorize live X API use, browser posting, auth mutation, Telegram,
or any external send.

Accepted decisions:

- `approve_publish` with approved post ids;
- `revise`;
- `park`;
- `reject`.

Bind the decision to the validated publish packet fingerprint. If the packet
changes after approval, require a new final gate.
