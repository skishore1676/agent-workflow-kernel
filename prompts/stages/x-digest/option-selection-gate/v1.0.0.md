# X Digest Option Selection Gate

Pause for Suman to select which options should become draft posts. Selection
only authorizes drafting. It does not authorize public posting, auth use, X API
calls, browser mutation, Telegram, or any external send.

Accepted decisions:

- `draft_selected` with selected option ids;
- `revise_options`;
- `park`.

Bind the decision to the option packet fingerprint so a later draft cannot
silently swap the source options.
