# Accept Source Approval

Bind the selected source packet to an explicit Suman approval receipt before
Ivy starts drafting. Treat ambiguous, stale, unchecked, or mismatched approval
evidence as `blocked`; do not infer approval from general comments, read-clear
checkboxes, or unrelated Blackboard state.

Return `selected` only when the source packet id, project id, gate, and intended
editorial scope are all visible in the input evidence. Record the approval
receipt id and any residual uncertainty for the next stage.
