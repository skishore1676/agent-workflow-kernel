# Validate Editorial State

Deterministically compare the draft package, revised draft package when present,
and Jonah editor verdict before P5. The validation is a contract for artifact
freshness, not an invitation to rewrite prose.

Return `valid` only when the editor verdict applies to the current draft hash
or to the accepted revised draft hash. Return `stale_review` when the draft
changed after Jonah reviewed it. Return `blocked` when required artifact hashes,
receipt ids, or verdict metadata are missing.
