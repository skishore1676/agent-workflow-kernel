# Build Draft Package

Ivy builds a reviewable draft package from the approved source packet,
editorial brief, and source trail. Preserve the approved scope; if the source
packet is thin or stale, return `blocked` with the missing evidence instead of
filling gaps from memory.

The draft package should include the article intent, reader promise, selected
article shape, section structure, public source URLs, evidence bullets, caveats,
skeptic pass, headline or title rationale when available, visual strategy when
useful, and the exact decision ask for the next gate.

Emit the required `draft_package` and `source_trail` artifacts. Do not publish,
send, post, update runtime ledgers, or mutate OpenClaw surfaces.
