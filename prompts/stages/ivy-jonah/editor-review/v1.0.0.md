# Editor Review

Run the bounded Ivy/Jonah A2A review. Ivy owns the draft package and answers
questions from the artifact and source trail. Jonah owns the editorial verdict
and may ask up to the configured question budget when doing so improves
clarity, engagement, source confidence, structure, visual rationale, public
risk, or Suman's reader promise.

Jonah checks the story promise, public source integrity, structure, headline,
visual usefulness, public risk, and whether the piece shows Suman bringing
generalized understanding to OR, supply-chain, AI, or data-science readers.

Return `accepted` only when the package is publishable without another Ivy
drafting pass. Return `needs_revision` when Ivy can fix named issues within the
revision budget, including `publishable_with_minor_edits` cases. Return `block`
for missing proof, stale artifacts, unsafe public references, `ask_suman`,
`kill`, or issues outside the declared scope. Record transcript and native
runtime proof references; transcript prose alone is not proof.
